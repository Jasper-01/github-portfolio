Pixel Exploration Game
A pixel exploration game that uses Unity Engine alongside C# coding language.
Project Details / Background
This project is a free to choose project. For our group, we decided to make an pixel exploration game. We decided to use Unity, but we create our own assetss.
I am specifically in charge of player behaviour, such as animation for idle, walk, and idle. I am also in charge of designing the game in terms of maps and quizzes, as I have the most experience on games similar to this. This is still an ongoing project. However, we are doing well and we are progressing smoothly.
This has certainly been a challenge for all of us as we have never used C# before, let along the Unity Enginge. Furthermore, most examples online use Unity3D instead of Unity2D, but we are using the latter as it is more convenient and makes more sense to our project. Though we have challenges, we discuss problems we are facing and our progression every Wedneday during class time. We have also learned to read documentation on IDEs, Engines, and Github as a whole for us to progress smoothly together.
